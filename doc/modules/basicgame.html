start (initialstate)
Creates a state machine and hooks it into LÖVE callbacks. When this function is called, a state machine will be created and initialstate will be pushed onto the top of the stack. This function will then set the LÖVE callbacks listed in gamestate.callbacks , plus love.load.

These events will be forwarded to the state machine, which in turn will dispatch them to the appropriate game state.

Parameters:

  • initialstate gamestate the state to start the basicgame with
