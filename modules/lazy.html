Module lazy

Implements lazily initialized values.

Lazy values are not produced until they are first requested. This allows you to delay a potentially expensive computation until the moment it is needed.

Usage:

    local filenames = lazy.new(function()
  return love.filesystem.enumerate('images/')
end)
...
for file in ipairs(filenames:get()) do
  local im = love.graphics.newImage(file)
  ...
end
