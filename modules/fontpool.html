Module fontpool

Facilitates the reuse of fonts instead of creating new ones.

DEPRECATED: Prefer an abstraction based on cache instead. The current functionality in this module may be duplicated with the following line of code:

fontpool = require('hug.cache').new(love.graphics.newFont)

This module will be removed in the future.
