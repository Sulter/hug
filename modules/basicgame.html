start (initialstate [, withexts=false])
Creates a state machine and hooks it into LÖVE callbacks. When this function is called, a state machine will be created and initialstate will be pushed onto the top of the stack. This function will then set the following callbacks:

  • love.load
  • love.draw
  • love.update
  • love.keypressed
  • love.keyreleased
  • love.mousepressed
  • love.mousereleased

These events will be forwarded to the state machine, which in turn will dispatch them to the appropriate game state.

Parameters:

  • initialstate gamestate the state to start the basicgame with
  • withexts bool If true, hug extensions will be installed into global tables. DEPRECATED: Prefer installing the extensions yourself, or require them and use them under a different name.
